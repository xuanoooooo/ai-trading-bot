### ✨ 新增功能

1. **自定义API完整支持**
   - 支持任何兼容OpenAI格式的API服务
   - 支持Ollama本地部署
   - 支持vLLM、LocalAI等自建服务

2. **完善的配置文档**
   - 新增 `.env.example` 配置模板
   - 详细的自定义API配置说明

3. **优化的错误提示**
   - 更友好的API Key检测提示
   - 明确指出自定义API配置方法

### 📝 配置示例

**Ollama本地部署**:
```json
{
  "ai_config": {
    "provider": "ollama",
    "model": "llama3.1:70b",
    "api_base": "http://localhost:11434/v1",
    "api_key_env": "CUSTOM_API_KEY"
  }
}
```

**.env 配置**:
```bash
CUSTOM_API_KEY=dummy-key
```
