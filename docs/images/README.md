# 看板截图说明

请将你的看板截图命名为 `dashboard-preview.png` 并放在此目录。

## 如何截图

1. 启动交易程序和看板
2. 访问 http://localhost:5000
3. 截取完整的看板界面
4. 保存为 `dashboard-preview.png`
5. 放到这个目录

## 截图建议

- 确保包含所有主要模块：账户总览、当前持仓、AI决策日志
- 使用浅色背景或深色背景都可以
- 建议分辨率：1920x1080 或更高
- 格式：PNG（支持透明背景）

## 示例位置

```
docs/images/dashboard-preview.png
```

完成后，README中的图片就会正常显示。
