# 启动AI多币种投资组合管理系统

## 方式1：使用tmux后台运行（推荐）

### 启动新会话
```bash
cd /root/duobizhong
tmux new -s portfolio
python3 src/core/portfolio_manager.py
```

### 常用tmux命令
- **断开会话**: `Ctrl+b` 然后按 `d`
- **重新连接**: `tmux attach -t portfolio`
- **查看所有会话**: `tmux ls`
- **停止程序**: `Ctrl+c` 然后 `exit`
- **杀死会话**: `tmux kill-session -t portfolio`

---



## 方式2：一条命令启动（推荐使用启动脚本）
```bash
cd /root/duobizhong
./scripts/start_portfolio.sh
```

或者手动创建后台会话：
```bash
tmux new -s portfolio -d
tmux send-keys -t portfolio "cd /root/duobizhong && python3 src/core/portfolio_manager.py" C-m
```

---

## 监控和管理

### 查看实时日志
```bash
# 方式1：如果在tmux中
tmux attach -t portfolio

# 方式2：查看日志文件
tail -f /root/duobizhong/portfolio_manager.log
```

### 查看进程状态
```bash
ps aux | grep portfolio_manager
```

### 停止程序
```bash
# 方式1：进入tmux停止
tmux attach -t portfolio
# 然后 Ctrl+c

# 方式2：杀死进程
pkill -f portfolio_manager.py
```

---

## 注意事项

1. **首次运行**：会自动创建 `portfolio_stats.json` 统计文件
2. **环境变量**：确保 `/root/duobizhong/.env` 文件配置正确
3. **日志文件**：程序会自动创建 `portfolio_manager.log`
4. **统计数据**：定期备份 `portfolio_stats.json`
5. **测试模式**：首次运行建议设置 `test_mode=True`
