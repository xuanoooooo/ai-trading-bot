### 🚀 快速开始（Ollama示例）

**1. 配置环境变量** (`.env`):
```bash
CUSTOM_API_KEY=dummy-key
BINANCE_API_KEY=your-binance-key
BINANCE_SECRET=your-binance-secret
```

**2. 配置AI模型** (`config/coins_config.json`):
```json
{
  "ai_config": {
    "provider": "ollama",
    "model": "llama3.1:70b",
    "api_base": "http://localhost:11434/v1",
    "api_key_env": "CUSTOM_API_KEY"
  }
}
```

**3. 启动交易**:
```bash
./start_portfolio.sh
```

就这么简单！✅
