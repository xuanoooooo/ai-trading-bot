📋 AI交易机器人 - 详细配置说明
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

本文档详细说明如何配置和使用AI交易机器人

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
一、系统要求
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Windows系统：
  - Windows 7/8/10/11 (推荐Windows 10及以上)
  - Python 3.8 或更高版本
  - 至少 2GB 可用内存
  - 稳定的网络连接

Linux系统：
  - Ubuntu 18.04+, Debian 10+, CentOS 7+, 或其他主流发行版
  - Python 3.8 或更高版本
  - 至少 2GB 可用内存
  - 稳定的网络连接

macOS系统：
  - macOS 10.14+ (Mojave或更新版本)
  - Python 3.8 或更高版本
  - 至少 2GB 可用内存
  - 稳定的网络连接

二、安装Python（如未安装）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Windows用户：
  1. 访问：https://www.python.org/downloads/
  2. 下载最新的Python 3.x版本（推荐3.11）
  3. 运行安装程序
  4. ⚠️ 重要：勾选 "Add Python to PATH" 选项
  5. 点击"Install Now"完成安装
  6. 验证安装：打开命令提示符，输入 python --version

Linux用户：
  Ubuntu/Debian:
    sudo apt-get update
    sudo apt-get install python3 python3-pip
  
  CentOS/RHEL:
    sudo yum install python3 python3-pip
  
  验证安装：
    python3 --version

macOS用户：
  方法1 - 使用Homebrew（推荐）:
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    brew install python3
  
  方法2 - 官方安装包:
    访问 https://www.python.org/downloads/mac-osx/
    下载并安装
  
  验证安装：
    python3 --version

三、获取API密钥
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. DeepSeek API密钥
   ──────────────────────────────────────────────────────────
   
   步骤：
   ① 访问：https://platform.deepseek.com/
   ② 点击"注册"创建账户（支持邮箱、手机号）
   ③ 登录后点击左侧"API Keys"
   ④ 点击"Create API Key"创建新密钥
   ⑤ 为密钥命名（例如：Trading Bot）
   ⑥ 复制生成的密钥（格式：sk-xxxxxx...）
   ⑦ ⚠️ 重要：密钥只显示一次，请立即保存
   
   充值：
   - 点击"充值"进行账户充值
   - 建议充值：50-100元人民币
   - 支付方式：支付宝、微信支付
   - 实际消费：每次AI调用约0.01-0.05元，100元可用很久
   
   费用说明：
   - 按实际调用量计费
   - 10分钟调用一次，每天约144次
   - 每次调用成本：0.02-0.05元
   - 每天成本：约3-7元
   - 100元充值可使用15-30天

2. 币安API密钥
   ──────────────────────────────────────────────────────────
   
   步骤：
   ① 访问：https://www.binance.com
      （美国用户访问：https://www.binance.us/）
   
   ② 注册并完成身份认证（KYC）
   
   ③ 登录后：
      - 点击右上角头像
      - 选择"API管理"
      - 点击"创建API"
   
   ④ 选择API类型：
      - 推荐选择"系统生成"
      - 输入API标签（例如：AI Trading Bot）
   
   ⑤ 完成安全验证（邮箱/手机验证码、谷歌验证码等）
   
   ⑥ 复制并保存：
      - API Key（公钥）
      - Secret Key（私钥）
      ⚠️ Secret Key只显示一次，请立即保存
   
   ⑦ 配置API权限：
      - ✅ 勾选"启用阅读"
      - ✅ 勾选"启用合约交易"
      - ⚠️ 不要勾选"启用提现"（安全考虑）
   
   ⑧ （可选但强烈推荐）设置IP白名单：
      - 点击"编辑限制"
      - 选择"限制访问可信的IP"
      - 添加您的服务器IP地址
   
   安全建议：
   - 🛡️ 强烈建议使用子账户而非主账户
   - 🛡️ 每个交易币种使用独立的子账户
   - 🛡️ 定期轮换API密钥（每月更换）
   - 🛡️ 不要在多个设备上使用同一API密钥

3. 创建币安子账户（强烈推荐）
   ──────────────────────────────────────────────────────────
   
   为什么要用子账户：
   - ✅ 风险隔离：主账户资金不受影响
   - ✅ 独立管理：每个币种一个子账户
   - ✅ 更安全：即使API泄露也不会影响主账户
   - ✅ 便于监控：交易记录清晰明了
   
   创建步骤：
   ① 登录币安主账户
   ② 进入"子账户管理"
   ③ 点击"创建子账户"
   ④ 设置子账户邮箱和密码
   ⑤ 为子账户转入资金（建议先小额测试）
   ⑥ 在子账户中创建API密钥（同上述步骤）

四、配置.env文件
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 找到.env文件
   - 文件位置：项目根目录下的 .env 文件
   - 如果看不到：需要显示隐藏文件
     Windows: 文件资源管理器 → 查看 → 勾选"隐藏的项目"
     Mac: Finder → Command+Shift+. (点)

2. 编辑.env文件
   - Windows: 右键 → 打开方式 → 记事本
   - Mac: 右键 → 打开方式 → 文本编辑
   - Linux: nano .env 或 vim .env

3. 填写API密钥
   
   找到这三行：
   DEEPSEEK_API_KEY=
   BINANCE_API_KEY=
   BINANCE_SECRET=
   
   在等号后面填入您的密钥：
   DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
   BINANCE_API_KEY=abcdefghij1234567890ABCDEFGHIJ
   BINANCE_SECRET=XyZ123aBc456DeF789GhI012JkL345MnO678
   
   ⚠️ 注意事项：
   - 不要删除等号(=)
   - 不要添加引号("")或单引号('')
   - 不要在等号前后添加空格
   - 确保每个密钥在单独一行
   - 密钥末尾不要有空格或换行

4. 保存文件
   - Windows: Ctrl+S 保存
   - Mac: Command+S 保存
   - Linux: Ctrl+X，然后Y，然后Enter

5. 验证配置
   配置完成后，.env文件应该类似：
   
   DEEPSEEK_API_KEY=sk-1234567890abcdef...
   BINANCE_API_KEY=ABCD1234EFGH5678...
   BINANCE_SECRET=xyz987abc654def321...
   LOG_LEVEL=INFO
   MAX_RETRIES=3
   REQUEST_TIMEOUT=30

五、币安账户设置
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 设置持仓模式（必须！）
   
   ⚠️ 非常重要：必须设置为"单向持仓模式"
   
   Web端设置：
   ① 登录币安
   ② 进入"合约交易"
   ③ 点击右上角"设置"图标
   ④ 选择"持仓模式"
   ⑤ 选择"单向持仓"
   ⑥ 确认修改
   
   App端设置：
   ① 打开币安App
   ② 进入"合约"
   ③ 点击左上角"设置"
   ④ 选择"持仓模式"
   ⑤ 选择"单向持仓"
   ⑥ 确认修改
   
   ⚠️ 如果使用"双向持仓模式"，程序将无法正常工作！

2. 资金划转
   
   将资金转入合约账户：
   ① 主账户 → 资金账户 → 合约账户
   ② 建议先转入小额资金测试（例如：100-500 USDT）
   ③ 测试成功后再增加资金

3. 杠杆设置
   
   默认杠杆：3倍
   - 程序会自动设置杠杆
   - 您也可以在币安界面手动设置
   - 建议不要使用过高杠杆（风险大）

八、常见问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q1: 提示"API密钥无效"
A: 检查.env文件中的密钥是否正确复制，没有多余空格

Q2: 提示"网络连接失败"
A: 检查网络是否能访问币安（大陆IP需要特殊网络）

Q3: 提示"权限不足"
A: 检查币安API是否开启了"合约交易"权限

Q4: 提示"ModuleNotFoundError"
A: Python依赖未安装，重新运行"一键安装"脚本

Q5: 程序启动后没有任何输出
A: 检查.env文件是否在正确位置，密钥是否填写

Q6: AI一直决策HOLD不交易
A: 正常现象，AI会等待合适的时机再入场

Q7: 如何修改交易币种
A: 需要编辑 src/deepseekBNB.py 文件中的配置

Q8: 如何查看历史日志
A: 查看项目目录下的 bnb_trader.log 文件

九、风险提示
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️ 重要风险提示：

1. 市场风险
   - 加密货币价格波动剧烈
   - AI决策不保证盈利
   - 可能产生亏损

2. 技术风险
   - 网络中断可能导致无法交易
   - API故障可能影响程序运行
   - 程序bug可能导致意外行为

3. 资金安全
   - 建议使用子账户
   - 建议从小额开始测试
   - 不要投入无法承受损失的资金

4. 使用建议
   - 定期检查程序运行状态
   - 定期检查账户余额
   - 设置止损策略
   - 不要过度依赖AI
