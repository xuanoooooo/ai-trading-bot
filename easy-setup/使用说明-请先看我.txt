━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 系统要求
   ✓ Python 3.8 或更高版本
   ✓ 稳定的网络连接
   ✓ Windows 7+ / Linux / macOS 10.14+

2. API密钥要求
   ✓ DeepSeek API密钥（需充值）
   ✓ 币安API密钥（需开启"合约交易"权限）
   ✓ 币安账户设置为"单向持仓模式"

3. 网络要求
   ⚠️ 大陆IP和美国IP无法直接访问币安
   ⚠️ 需要自行解决网络问题

4. 安全建议
   🛡️ 强烈建议使用币安子账户
   🛡️ 每个币种使用独立子账户
   🛡️ 建议先用小额资金测试

5. 风险提示
   ⚠️ 加密货币交易有风险
   ⚠️ AI决策不保证盈利
   ⚠️ 请根据自身风险承受能力使用

🔧 Windows系统详细步骤
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

步骤1：检查Python（如未安装请先安装）
  - 按Win+R → 输入cmd → 输入python --version
  - 如果提示错误，请下载安装：https://www.python.org/downloads/
  - ⚠️ 安装时必须勾选"Add Python to PATH"

步骤2：配置.env文件
  - 找到项目文件夹中的.env文件
  - 右键 → 打开方式 → 记事本
  - 找到以下3行并填入你的密钥：
    DEEPSEEK_API_KEY=在这里填入你的DeepSeek密钥
    BINANCE_API_KEY=在这里填入你的币安API密钥
    BINANCE_SECRET=在这里填入你的币安Secret密钥
  - 按Ctrl+S保存

步骤3：首次安装
  - 打开scripts文件夹
  - 双击"Windows系统一键安装.bat"
  - 等待安装完成（需要1-3分钟）

步骤4：启动程序
  - 双击"Windows系统一键启动.bat"
  - 程序窗口会保持打开
  - 看到"BNB自动交易机器人启动成功"即成功

步骤5：停止程序
  - 直接关闭程序窗口
  - 或按Ctrl+C

🐧 Linux系统详细步骤
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

步骤1：解压并进入目录
  cd ~/Downloads  # 或你的下载目录
  unzip ai-trading-bot-easy-setup.zip
  cd ai-trading-bot-easy-setup

步骤2：配置.env文件
  nano .env  # 或使用vim .env
  # 填入你的API密钥，然后按Ctrl+X，Y，Enter保存

步骤3：给脚本执行权限
  chmod +x scripts/*.sh

步骤4：首次安装
  bash scripts/Linux系统一键安装.sh
  # 等待安装完成（需要1-3分钟）

步骤5：启动程序
  bash scripts/Linux系统一键启动.sh
  # 程序开始运行

步骤6：后台运行（可选）
  nohup bash scripts/Linux系统一键启动.sh > trading.log 2>&1 &
  # 查看后台进程：ps aux | grep deepseekBNB
  # 停止后台进程：pkill -f deepseekBNB

❓ 常见问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q: 双击.bat文件一闪而过？
A: 右键.bat文件 → 编辑，检查是否有错误提示

Q: 提示"找不到Python"？
A: 需要先安装Python，并确保勾选"Add Python to PATH"

Q: 提示"API密钥无效"？
A: 检查.env文件中密钥是否正确，没有多余空格

Q: Linux下提示"Permission denied"？
A: 运行：chmod +x scripts/*.sh

Q: 如何查看程序日志？
A: 查看项目目录下的 bnb_trader.log 文件

Q: 如何修改交易币种？
A: 需要编辑 src/deepseekBNB.py 文件（需要一定编程基础）
