🤖 AI自动交易机器人 - 5分钟快速开始指南
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️ 重要提示：请仔细阅读本指南，确保正确配置后再启动！

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第一步：获取API密钥（最重要！）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 获取 DeepSeek AI 密钥：
   ① 访问：https://platform.deepseek.com/
   ② 注册账户并登录
   ③ 点击"API Keys"创建新密钥
   ④ 复制密钥（格式类似：sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx）
   ⑤ 💰 充值建议：充值50-100元即可使用很长时间

2. 获取币安API密钥：
   ① 访问：https://www.binance.com （注意：美国用户使用 binance.us）
   ② 登录账户 → 个人中心 → API管理
   ③ 创建新的API密钥
   ④ 复制 API Key 和 Secret Key
   ⑤ ⚠️ 重要权限设置：
      - ✅ 启用"合约交易"权限
      - ✅ 启用"读取"权限
      - ⚠️ 关闭"提现"权限（安全考虑）
   ⑥ 🛡️ 建议设置IP白名单（限制API只能从您的服务器访问）

3. 🛡️ 安全建议：
   - 强烈建议使用币安子账户进行交易
   - 每个币种用一个独立子账户，有效隔离风险
   - 不要在主账户上直接使用

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第二步：配置API密钥
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 找到解压后的文件夹中的 ".env" 文件
   
2. 使用记事本打开 .env 文件（右键 → 打开方式 → 记事本）

3. 填写您的API密钥：

   DEEPSEEK_API_KEY=sk-你的DeepSeek密钥
   BINANCE_API_KEY=你的币安API密钥
   BINANCE_SECRET=你的币安Secret密钥

4. 保存文件（Ctrl+S）

⚠️ 注意：
   - 不要删除等号(=)
   - 不要添加引号或空格
   - 确保每个密钥都在单独一行

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第三步：启动程序
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Windows用户：
  1. 双击运行 scripts 文件夹中的"Windows系统一键安装.bat"（首次使用）
     → 等待依赖安装完成（需要联网，约1-3分钟）
  
  2. 双击运行"Windows系统一键启动.bat"
     → 程序开始运行，看到AI分析输出即成功！

Linux/Mac用户：
  1. 打开终端，进入程序目录
     cd /path/to/ai-trading-bot
  
  2. 运行安装命令（首次使用）：
     chmod +x scripts/*.sh
     bash scripts/Linux系统一键安装.sh
  
  3. 运行启动命令：
     bash scripts/Linux系统一键启动.sh
  
  4. 后台运行（可选）：
     nohup bash scripts/Linux系统一键启动.sh > trading.log 2>&1 &
  
  5. 查看后台进程：
     ps aux | grep deepseekBNB
  
  6. 停止后台进程：
     pkill -f deepseekBNB

  ✅ 看到 "当前账户余额" 信息

常见错误：
  ❌ "API密钥无效" → 检查.env文件中的密钥是否正确
  ❌ "网络连接失败" → 检查网络是否能访问币安（需要特殊网络）
  ❌ "权限不足" → 检查币安API是否开启合约交易权限
  ❌ "ModuleNotFoundError" → 重新运行"一键安装.bat"
